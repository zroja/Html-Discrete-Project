5 properties of Relations

1.Reflexive= if and only if EQUAL (a,a) of set A
2.Irreflexive= if and only if NOT EQUAL (a,a) of set A
3.Symmetric= if and only if in set A, (a,b) = (b,a)
4.Antisymmetric= if and only if in set A, (a,b) NOT EQUAL (b,a)
5.Transitive= if and only if (a,b), (b,c), (a,c) in set A
