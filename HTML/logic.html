Logic

Propositional Logic

What is a proposition?

A proposition is a statement that is either true or false.
The truth value of a proposition = T or F

propositional variables = p or q
Negation is the NOT operation, it reverses the truth value.
Below is a truth table of negation:

p p'
T F
F T

Conjunction and Disjunction



Conjunction(AND) is true if BOTH operands are true. Denote by ^.

p q p^q
T T T
T F F
F T F
F F F

Disjunction(OR) is true if EITHER operand is true including both.

p q pVq
T T T
T F T
F T T
F F F

Exclusive(OR) is true if EITHER operand is true excluding both.

p q p+q
T T F
T F T
F T T
F F F

Conditional: This table signifies if p then q.

p q p-->q
T T T
T F F
F T T
F F T

Biconditional:This table signifies p if and only if q.

p q p<-->q
T T T
T F F
F T F
F F T
