When is something not a function?

NOT A FUNCTION WHEN ONE INPUT HAS TWO DIFFERENT OUTPUTS! NOT POSSIBLE

3 Function Properties

1. Surjection= every output must have AT LEAST one input
2. Injection = no two inputs can have the same output
3. Bijection = each output must have exactly one input

Pigeonhole Principle

Identity, Inverse, and Composite Functions

Identity= the input is unchanged, it is identically the same as it was.


Inverse= the domain and the range of the function are flipped


Composite Function= When you evaluate a function with another function.
