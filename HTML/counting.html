Counting

Elementary Rules of Counting

Multiplication Rule: We have 2 choices C1 and C2, which C1 having set
O1 possible outcomes and C2 with O2 possible outcomes, and |O1|= n2 and |O2|=n2
so then the total number of possible outcomes for C1 and C2 are n1*n2...

For example:

Choices:
C1= choose the jeans style
C2= choose the jeans color
Outcomes:
O1={standard fit, loose fit, boot fit, slim fit}
O2= {black, blue}
Cardinalities
n1= |O1|= 4 n2=|C2|=2
multiplication rule= n1*n2= 4*2=8= total number of outcomes


Addition Rule : We have 2 choices C1 and C2, which C1 having set
O1 possible outcomes and C2 with O2 possible outcomes, and |O1|= n2 and |O2|=n2 so
then the total number of possible outcomes for C1 and C2 are n1+n2...

For example, You need to purchase one shirt of any kind.
The store has five short sleeve shirts and eight long sleeve shirts.
How many possible ways are there to choose a shirt?
Solution: 8+5=13
Roughly Speaking: Addition = "OR" and Multiplication = "AND"

Permutations and Combinations

Permutation = Order matters, cannot reuse objects
Formula= n!/(n − r)


Combination = Order does not matter, cannot reuse objects
Formula = n!/(n − r)!r!
