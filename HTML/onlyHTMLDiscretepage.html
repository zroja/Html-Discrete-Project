What is a set?

A set is a collection of objects(the members of elements of the set)For example, {a,b,c,d,e}

In a set, order does not matter and repetitions do not matter

Set Operations

Cardinality: The number of elements in a set

Union: The set of all elments belonging to EITHER of two given sets.

Intersection: The set of all elements belonging to BOTH of two given sets. Disjoint = intersection empty

Difference : set of elements belonging to one set, but not another.

Complement : Set of all elements from the universal set that DO NOT belong to a given set

Cartesian Product : set of all ordered pairs from two given sets

Power Set: set of subsets of a given set

Here is an example of the principle of inclusion/exclusion:

Principle of Inclusion or Exclusion



Let K, P, T represent the sets of people who like ketchup,
pickles, and tomatoes on their hamburgers. Suppose that
|K| = 20 |P| = 30 |T| = 45
|K ∩P| = 10 |K ∩T| = 12 |P ∩T| = 13
|K ∩P ∩T| = 8

Then, |K ∪P ∪T| = |K| + |P| + |T|−
|K ∩P| − |K ∩T| − |P ∩T|+ |K ∩P ∩T| = 20 + 30 + 45 − 10 − 12 − 13 + 8 = 68
